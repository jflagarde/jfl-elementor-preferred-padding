Use the up and down arrow to go through your favorite padding values.

== Description ==

On padding and margin settings, you can use your up and down arrows to go through your favorite values. Here they are setup as [0,10,20,40,80,160].

== Installation ==

1. Upload the content of `jfl-elementor-preferred-padding.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
